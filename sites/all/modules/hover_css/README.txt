---DESCRIPTION---

Hover is a collection of CSS3 powered hover effects to be applied to links,
buttons, logos, SVG, featured images and so on.

---Dependencies---

Libraries
Hover in libraries/hover
(can be downloaded from https://github.com/IanLunn/Hover)

---Module usage---

This module allows you to implement Hover css properties in you js/css.
So once you have this module enabled, you can use its properties.
Eg : in your Js you can write : $('#yourElement').addClass('hvr-grow');

---INSTALLATION AND CONFIGURATION---
1. Download and ADD the library
from https://github.com/IanLunn/Hover in libraries/hover

2. Install the module as usual.

3. In your js file you can write the code.
Library usage : https://github.com/IanLunn/Hover/blob/master/README.md
