-- Page Transition --

-- SUMMARY --

This module will helps to add page transitions to the drupal pages.

-- INSTALLTION --

1) Copy page_transition directory to your modules directory.
2) Download Animsition library from
   https://github.com/blivesta/animsition
   and put it in libraries folder.
3) Animsition library folder name
   should be 'animsition'.
4) Enable the module at module configuration page.
5) Configure Page Transition in
   admin/config/page_transition/manage.

-- FEATURES --

1) Easy integration with Animistion page transition plugin.
2) Detailed configuration page to manage different attributes of
   page transition.
