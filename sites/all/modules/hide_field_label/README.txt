INTRODUCTION
------------

 * Hide Field Label module provides a possibility to hide field labels on widget
   forms.

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/hide_field_label

 * Currently supported widget types are text_textfield, text_textarea,
   text_textarea_with_summary, number, email_textfield. This list can be
   extended with hook_hide_field_label_widgets_alter().

REQUIREMENTS
------------

 * This module requires Field module enabled.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. See:
   https://www.drupal.org/documentation/install/modules-themes/modules-7
   for further information.


CONFIGURATION
-------------

 * Just go to the field instance settings page and put a tick on "Hide label
   for this field" checkbox, then click "Save settings" button
