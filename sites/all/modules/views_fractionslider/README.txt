
/* DESCRIPTION */
FractionSlider is a jQuery plugin which adds a fully customization element 
slider to your website. FractionSlider is HTML5 multi-layer and parallax 
slideshow.

/* INSTALATION */
>> Download Views FractionSlider. It requires jQuery Update and Libraries API
 module.
https://drupal.org/project/views_fractionslider
https://www.drupal.org/project/Jquery_update
https://www.drupal.org/project/libraries

>> Now Go to /admin/build/modules, enable module, "FractionSlider".

>> FractionSlider is ready to use. Check ready-made Slider block named
"FractionSlider" Under /admin/structure/block and its settings can be change
from inside block at 
/admin/structure/block/manage/fractionslider/fractionslider/configure
